# DSP UPAC/LPAC Simulator

웨이퍼 형상 시뮬레이션을 위한 React 기반 웹 애플리케이션입니다.

## 기능

- UPAC/LPAC 파라미터 조정을 통한 웨이퍼 형상 시뮬레이션
- 실시간 히트맵 시각화
- 웨이퍼 형상 변화 그래프
- 반응형 디자인

## 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm start
```

3. 브라우저에서 `http://localhost:3000`으로 접속

## 빌드

프로덕션 빌드:
```bash
npm run build
```

## 기술 스택

- React 18
- Tailwind CSS
- JavaScript (ES6+)
